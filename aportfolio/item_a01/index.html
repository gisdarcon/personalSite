Internal Migration Around the GlobE (IMAGE) University of Leeds


 

 

 

Project Description

This project aims to investigate the way in which internal migration - the propensity to change residence within national borders - varies between countries around the world. More info at http://www.gpem.uq.edu.au/qcpr-image

The IMAGE Studio is an analytical software to compute internal migration measures and address key methodological issues. The source code and the binaries are available through the Github host services.

IMAGE Studio Framework

The IMAGE studio is organized as a set of linked systems: i) the data preparation system, ii) the spatial aggregation, iii) the internal migration indicators system, and iv) the spatial interaction modelling system.
